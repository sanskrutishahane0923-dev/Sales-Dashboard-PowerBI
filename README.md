# 📊 Sales Dashboard (Power BI)

## 📌 Project Overview
This project presents a Sales Dashboard built using Power BI to analyze business performance.

## 🎯 Objectives
- Analyze total sales, profit, and orders
- Identify top-performing regions and categories
- Track monthly sales trends

## 🛠 Tools Used
- Power BI
- Microsoft Excel

## 📈 Key Insights
- West region generated highest sales
- Electronics category contributed maximum revenue
- Sales showed variation across months
